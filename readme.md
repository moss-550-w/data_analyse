# 个人消费账单数据分析项目

## 项目概述

本项目基于微信支付账单流水数据，运用多种数据挖掘和统计建模方法，对个人消费行为进行全面分析。项目涵盖数据预处理、消费行为分群、消费趋势预测、消费类目关联分析、消费结构评估和消费影响因素分析六大模块，旨在揭示消费规律、识别优化空间，为理性消费提供数据支撑。

## 数据说明

### 数据来源
- **原始数据**：微信支付账单流水文件（2026-01-30至2026-04-30）
- **样本规模**：132笔有效交易记录
- **时间跨度**：2026年2月2日至2026年4月29日
- **消费类目**：超市购物、餐饮美食、交通出行、服饰美容、其他

### 数据预处理
- **去重处理**：剔除重复交易记录
- **异常值检测**：使用IQR和3σ方法识别异常值
- **数据标准化**：统一金额单位，规范时间格式
- **特征工程**：提取时间特征（月份、星期、时段等）和消费属性特征

---

## 分析模块

### 模块一：消费行为分群分析（PCA+K-Means）

**分析方法**：主成分分析（PCA）降维 + K-Means聚类

**核心发现**：
- 识别出2个主要消费行为群体
- 群体1（高频低额型）：以超市购物为主，日均消费7.45元，消费稳定
- 群体2（低频高额型）：以"其他"类大额消费为主，单笔258元，偶发性强

**业务价值**：为差异化消费管理策略提供依据

**输出文件**：
- `Clustering.py` - 聚类分析代码
- `result_analuse.md` - 详细分析报告

---

### 模块二：消费趋势预测（STL+SARIMA）

**分析方法**：STL时间序列分解 + SARIMA季节性预测

**核心发现**：
- 消费数据呈现明显的趋势性、季节性和残差成分
- 未来7天消费金额预测区间：50-65元
- 消费趋势整体平稳，无显著上升或下降趋势

**业务价值**：为未来消费规划和预算制定提供参考

**输出文件**：
- `Predict.py` - 预测模型代码
- `result_analyse.md` - 详细分析报告

---

### 模块三：消费类目关联分析（Apriori）

**分析方法**：Apriori关联规则算法

**核心发现**：
- 挖掘出10条消费关联规则
- 最强关联：{超市购物+其他/交通出行}→{餐饮美食}（提升度4.36）
- 消费呈现"购物-餐饮"场景化联动特征

**业务价值**：识别消费组合规律，优化消费决策

**输出文件**：
- `Association.py` - 关联规则挖掘代码
- `result_analyse.md` - 详细分析报告

---

### 模块四：消费结构评估（洛伦兹曲线+基尼系数+ANOVA）

**分析方法**：洛伦兹曲线 + 基尼系数 + 单因素方差分析

**核心发现**：
- 基尼系数：0.570（相对集中，需关注）
- 消费高度集中于"其他"类（57.0%）和超市购物（30.2%）
- 星期和月份因素对消费金额影响不显著（p>0.05）

**业务价值**：评估消费结构合理性，识别优化方向

**输出文件**：
- `Structural_Assessment.py` - 结构评估代码
- `resuly_analyse.md` - 详细分析报告

---

### 模块五：消费影响因素分析（随机森林+DID）

**分析方法**：随机森林特征重要性 + 双重差分法（DID）

**核心发现**：
- TOP3影响因素：小时（34.3%）、消费类目（30.7%）、时段（22.4%）
- 发薪日效应不显著（DID系数=-29.02，p=0.415）
- "其他"类消费83.3%集中在发薪日附近，但因果关系不明

**业务价值**：识别消费驱动因素，制定精准控制策略

**输出文件**：
- `Influencing_factors.py` - 影响因素分析代码
- `result_analyse.md` - 详细分析报告

## 核心结论汇总

### 1. 消费行为特征
- **消费结构**：呈现"双核驱动"特征，"其他"类（57.0%）和超市购物（30.2%）占主导
- **时间分布**：消费金额无明显周期性波动，时间稳定性较高
- **类目差异**：各类目消费金额差异显著（258元 vs 7.45元，差距34.6倍）

### 2. 消费驱动因素
- **首要因素**：具体时间点（小时、时段）决定消费金额，合计贡献56.7%
- **次要因素**：消费类目属性，贡献30.7%
- **非显著因素**：发薪日、周末等周期性因素对消费金额影响不显著

### 3. 消费关联规律
- **场景化联动**：存在"超市购物+其他→餐饮美食"的强关联（提升度4.36）
- **双向关联**：部分消费类目间存在相互促进的双向关联关系

### 4. 预测与预警
- **短期预测**：未来7天日均消费约50-65元
- **预警机制**：建议在晚上时段（18-22点）设置消费预警，控制非必要支出

## 业务应用建议

### 1. 优化消费结构
- **细化"其他"类消费**：建立大额消费记录机制，提高消费透明度
- **控制基尼系数**：目标控制在0.50以下，降低消费集中度
- **平衡类目支出**：适当增加餐饮、文化等类目的消费占比

### 2. 建立预算管理体系
- **时段预算**：针对早晨和晚上高频消费时段设置预算上限
- **类目预算**：按消费特征制定差异化预算（周预算、日预算、月预算）
- **分级预警**：建立黄色（50%）、红色（60%）两级预警机制

### 3. 强化消费控制
- **大额消费审批**：对单笔超过100元的非必要消费设置24小时冷静期
- **时段消费提醒**：在晚间消费高峰时段自动推送消费提醒
- **定期复盘**：每季度进行一次消费结构评估和调整

## 模型局限性与改进方向

### 当前局限
1. **样本量限制**：仅132笔记录，部分统计检验功效不足
2. **时间跨度短**：3个月数据，无法反映季节性变化
3. **类目粒度粗**："其他"类占比过高（57%），细分不足
4. **特征维度少**：缺少收入、储蓄等核心财务特征

### 改进方向
1. **数据积累**：持续收集数据，目标积累1年以上记录
2. **类目细化**：对"其他"类进行子类目拆分
3. **特征扩展**：引入收入、资产、负债等个人财务特征
4. **模型优化**：
   - 使用因果森林进行异质性因果效应分析
   - 引入深度学习模型提升预测精度
   - 构建实时消费预警系统
